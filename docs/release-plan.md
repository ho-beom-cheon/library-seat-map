# 릴리스 계획

이 문서는 `library-seat-map`의 릴리스 후보와 알려진 제한 사항을 관리합니다.

## v0.1.0 MVP 후보

- 서울 일부 공공도서관 기본정보 수집
- 열람실별 실시간 좌석정보 수집
- 도서관, 열람실, 좌석 스냅샷 DB 저장
- 지도 기반 도서관 조회
- 잔여 좌석과 사용률 기반 마커 상태 표시
- 최근 갱신 기준 상세 카드 표시
- 좌석, 거리, 사용률, 운영시간 기반 추천 API
- 공공 API 장애 시 마지막 정상 데이터 유지
- 위치정보 저장 없이 주변 도서관 조회

## 알려진 제한 사항

- 좌석 데이터가 제공되는 도서관만 추천할 수 있습니다.
- 운영현황 API는 MVP 핵심 로직에서 제외하고 예비 구현으로 둡니다.
- 좌석은 보장되지 않고 최근 갱신 기준 가능성으로만 제공합니다.
- 1차 대상 지역은 강남구 중심으로 시작합니다.

## 릴리스 후보 작성 기준

다음 변경은 릴리스 노트 후보에 포함합니다.

- 사용자에게 보이는 기능 추가 또는 변경
- API 응답 구조 변경
- 실행 방법, 환경변수, 배포 방식 변경
- 보안, 개인정보, 위치정보 처리 방식 변경
- 운영자 또는 유지보수자에게 영향을 주는 문서 변경

단순 오탈자 수정이나 내부 정리만 있는 변경은 릴리스 노트 후보에서 제외할 수 있습니다.
